# RDP ActiveX Control - Compilación e Instalación\n\nEste documento describe cómo compilar e instalar el control RDP ActiveX mejorado.\n\n## Requisitos Previos\n\n1. **Visual Studio Build Tools 2019 o superior** con:\n   - MSVC v142 compiler toolset\n   - Windows 10/11 SDK\n   - CMake tools\n\n2. **Node.js** (versión 16 o superior)\n\n3. **Python** (versión 3.8 o superior)\n\n## Pasos de Instalación\n\n### 1. Instalar Dependencias\n\n```bash\ncd native/rdp-activex\nnpm install\n```\n\n### 2. Compilar el Módulo Mejorado\n\n```bash\n# Opción 1: Usar la configuración avanzada\nnode-gyp configure --msvs_version=2019 -f binding-advanced.gyp\nnode-gyp build --msvs_version=2019 -f binding-advanced.gyp\n\n# Opción 2: Usar npm scripts (recomendado)\nnpm run build-advanced\n```\n\n### 3. Verificar la Compilación\n\n```bash\nnode test-advanced.js\n```\n\n## Estructura de Archivos\n\n```\nrdp-activex/\n├── src/\n│   ├── rdp_activex_basic.cpp      # Implementación básica (placeholder)\n│   └── rdp_activex_improved.cpp   # Implementación mejorada con ActiveX real\n├── binding.gyp                    # Configuración básica\n├── binding-advanced.gyp           # Configuración mejorada\n├── index.js                       # Manager de JavaScript\n└── package.json\n```\n\n## Características de la Implementación Mejorada\n\n### Control ActiveX Real\n- Utiliza el control RDP nativo de Windows (MsRdpClient)\n- Soporte para múltiples versiones del cliente RDP\n- Embebido completamente en la aplicación Electron\n\n### Gestión de Ventanas\n- Contenedor nativo de Windows para el control\n- Manejo automático de eventos de redimensionamiento\n- Limpieza adecuada de recursos COM\n\n### Configuración Avanzada\n- Resolución personalizable\n- Credenciales seguras\n- Configuración de parámetros avanzados\n\n## Solución de Problemas\n\n### Error: \"No se pudo crear la instancia del control ActiveX\"\n\n1. Verificar que el control RDP esté instalado:\n   ```bash\n   regsvr32 mstscax.dll\n   ```\n\n2. Ejecutar como administrador si es necesario\n\n3. Verificar que Windows tenga habilitado RDP\n\n### Error: \"RDP ActiveX manager not available\"\n\n1. Recompilar el módulo:\n   ```bash\n   npm run rebuild-advanced\n   ```\n\n2. Verificar que se esté cargando la versión correcta:\n   ```javascript\n   console.log(require('./native/rdp-activex/build/Release/rdp_activex_advanced.node'));\n   ```\n\n### Error de Compilación\n\n1. Limpiar y reconstruir:\n   ```bash\n   npm run clean\n   npm run build-advanced\n   ```\n\n2. Verificar Visual Studio Build Tools:\n   ```bash\n   npm config set msvs_version 2019\n   ```\n\n## Uso Programático\n\n```javascript\nconst RdpActiveXManager = require('./index');\n\nconst manager = new RdpActiveXManager();\nconst instanceId = manager.createInstance(parentWindowHandle);\n\n// Configurar conexión\nmanager.setServer(instanceId, 'server.example.com');\nmanager.setCredentials(instanceId, 'username', 'password');\nmanager.setDisplaySettings(instanceId, 1920, 1080);\n\n// Conectar\nmanager.connect(instanceId);\n\n// Limpiar al terminar\nmanager.cleanup();\n```\n\n## Scripts NPM Disponibles\n\n```json\n{\n  \"scripts\": {\n    \"build-basic\": \"node-gyp rebuild\",\n    \"build-advanced\": \"node-gyp rebuild -f binding-advanced.gyp\",\n    \"clean\": \"node-gyp clean\",\n    \"rebuild-advanced\": \"npm run clean && npm run build-advanced\",\n    \"test\": \"node test-basic.js\",\n    \"test-advanced\": \"node test-advanced.js\"\n  }\n}\n```\n\n## Notas de Desarrollo\n\n- La implementación usa COM (Component Object Model) de Windows\n- Requiere thread apartment STA (Single Threaded Apartment)\n- Maneja automáticamente la limpieza de recursos COM\n- Compatible con Electron mediante N-API\n\n## Próximos Pasos\n\nPara usar el control mejorado, actualiza el main.js para usar 'rdp_activex_advanced' en lugar de 'rdp_activex_basic'.\n
